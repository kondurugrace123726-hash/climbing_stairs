# climbing_stairs
Climbing Stairs using Python.

## 📌 Problem Statement
You are climbing a staircase. It takes 'n' steps to reach the top.

Each time you can either climb:
- 1 step
- 2 steps

Find the total number of distinct ways to reach the top.


## ⏱ Time Complexity
O(n)

## 💾 Space Complexity
O(1)
